# FreshCart app

This is the Expo (SDK 56) app for the FreshCart grocery demo — it builds the
Android APK, the website, and the web-only admin console from one codebase.

**Read the root [../../CLAUDE.md](../../CLAUDE.md) for full project context**:
architecture, conventions (₹ currency, no hardcoded URLs, zustand selector rule),
the monorepo + Vercel/EAS setup, and known gotchas (hoisting fix, no `npx expo install`).
